Client: Secret Games Company
Role: Illustration • UI • UX • Animation
Platform: Windows • Mac
Kim is an open world RPG set in colonial India. Play as Rudyard Kipling's ragamuffin hero, Kim, and roam hand-painted towns and procedurally generated countryside.

Working closely with the director I developed the games visual style, digitally painted the environments, designed the UI, and animated the characters.

Environments

Each of the 18 towns was designed to be a pocket sized version of the real thing, from the time of the British empire. I created assets to be used in the randomly generated countryside.
