Client: Sleeping Beast Games
Role: Illustration • UI • UX • Animation
Platform: Android • IOS
Blabyrinth is a local multiplayer mobile game where players attempt to escape a randomly generated maze. The game features cryptic puzzles and secret passages.

I developed the games visual style, assisted with game design, and animation for the trailer, as well as illustrated over 1000 game items, scenery and players.
