Disco Elysium Collage Mode

As lead UX designer at ZA/UM, I was tasked with designing a free expansion for Disco Elysium; Collage Mode, a tool for creating scenes with game assets.

Specifications

Collage Mode in Disco Elysium is designed to allow players to create and customize scenes using characters, environments, and effects from the game, enabling them to build diaramas.

This feature includes options to pose characters, apply filters, and frame their creations with various themes.

DE had a large amount of assets, which we also wanted to add to, stickers, effects and easter eggs.

Functionality
The player can move, rotate, scale all with mouse. controller inputs are mapped similarly, so that it could be used with a mouse, keys and controller seperately. move -> position next/previous (lateral motion) -> rotate zoom -> scale The player is able to favorite any game asset, to make it sticky on the left side of the asset drawer [interaction video]
